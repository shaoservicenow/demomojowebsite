🎨 DESIGN GUIDELINES:
====================

✅ DO:
- High contrast for readability
- Clean, professional interface
- Consistent DemoMojo theme colors
- Clear annotations/arrows
- Remove personal/sensitive data
- Use consistent styling

❌ DON'T:
- Blurry or low-quality images
- Cluttered interfaces
- Personal information visible
- Inconsistent colors/themes
- Too much text in screenshots
- Overly complex annotations

🚀 IMPLEMENTATION:
=================

Once screenshots are ready:
1. Place all files in docs-screenshots/ folder
2. Update docs.html to reference these images
3. Replace placeholder divs with <img> tags
4. Test on both desktop and mobile
5. Deploy to GitHub Pages

Example HTML structure:
<div class="screenshot-container">
  <img src="docs-screenshots/docs-overlay-demo.png" 
       alt="DemoMojo overlay in action" 
       class="docs-screenshot">
</div>
